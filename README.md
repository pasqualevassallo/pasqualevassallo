# Ciao, sono Pasquale Vassallo 👋
Ingegnere Informatico.

Benvenuti nel mio [portfolio](https://pasqualevassallo.github.io/portfolio/) di progetti universitari sviluppati durante il percorso di studi presso l'**Università degli Studi della Campania "Luigi Vanvitelli"**.

* **[Ingegneria del Software](https://github.com/pasqualevassallo/Rescue-Bite)**: Progettazione di una piattaforma per la gestione delle eccedenze alimentari (Ingegneria del Software).
* **[Sistemi Web e Basi di Dati](https://github.com/pasqualevassallo/MusicMatch)**: Web App per la gestione e prenotazione di lezioni musicali con database relazionale MySQL.
* **[Internet of Things](https://github.com/pasqualevassallo/Memory-Color-Game)**: Sistema integrato hardware-software (ESP32) ispirato a "Simon Says" con gestione tramite web dashboard.
* **[Protocolli di Sicurezza e Sistemi di Rete](https://github.com/pasqualevassallo/SpearPhishing-ARP-DNS-Spoofing)**: Simulazione di attacchi di ingegneria sociale, ARP Poisoning e DNS Spoofing con iniezione di Reverse TCP Shell.
* **[Metodi di Ottimizzazione](https://github.com/pasqualevassallo/ESA-SpOC2-Optimization)**: Progetto di ottimizzazione sviluppato per la Space Optimisation Challenge – Quantum Communications Constellations, proposta dall’European Space Agency (ESA).
* **[Multivariable Feedback Control](https://github.com/pasqualevassallo/Unstable-System-Control)**: Analisi, stabilizzazione e controllo di un sistema instabile di ordine 4. Include sintesi LQR, LQG e procedura LTR per il recupero della robustezza.
